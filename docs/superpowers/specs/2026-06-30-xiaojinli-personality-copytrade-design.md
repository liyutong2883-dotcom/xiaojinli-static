# 小金鲤人格化抄作业改版设计

日期：2026-06-30

## 背景

当前原型仍然更像传统金融 App：用户进入后先看到搜索、交易热点、热门主题、工具箱和作业列表。这个结构的问题是信息很多，但主线不够锋利。用户来到“小金鲤”的核心动机不是研究市场，而是降低决策成本：我这种人该看谁的作业、这个作业为什么赚钱、我能不能抄。

本次改版选择方案 B：重构信息架构。目标不是简单替换首页模块，而是把产品主线改成“趣味人格入口 + 个性化作业流 + AI 拆作业”。

## 产品定位

小金鲤不是行情工具，也不是投研终端。它更像一个低门槛、人格化、内容化的投资作业发现产品。

核心承诺：

> 先知道你是哪种投资人格，再给你看更适合你抄的作业。

产品应该给用户的感觉是“好玩、能看懂、有人带我拆”，而不是“专业、复杂、需要自己研究半天”。

## 目标

1. 把第一触点从行情信息改成五道题趣味测试。
2. 用动物人格降低用户理解成本，形成可传播、可记忆的人设标签。
3. 首页优先展示“今天谁赚了大钱”和“适合你的作业”，弱化传统行情热点。
4. 作业详情直接给结论、持仓组合和 AI 解释，减少层层点击。
5. 保留小金鲤 AI 作为随时提问入口。
6. 将工具箱从主流程移出，后续作为独立运营工具处理。

## 非目标

1. 本阶段不做真实交易、开户、下单或资产连接。
2. 本阶段不做真实风控适当性测评，只做趣味人格分类。
3. 本阶段不接真实行情或持仓接口，仍使用静态模拟数据。
4. 本阶段不建设完整社区，只预留“灵感/记录”入口。
5. 本阶段不把工具箱作为主导航重点。

## 核心用户流

```text
用户着陆
  -> 五道题趣味测试
  -> 动物人格画像
  -> 个性化作业首页
  -> 看作业：谁赚了大钱
  -> 拆作业：AI 解释持仓逻辑
  -> 抄作业 / 收藏 / 写灵感
  -> 不确定时问小金鲤
```

首次进入时，用户应优先进入测试流程。已有画像用户可以直接进入首页，但首页顶部仍要展示当前人格和推荐理由。

## 动物人格设计

动物人格用于降低理解成本，不等同于严肃风险测评。

| 人格 | 核心倾向 | 推荐作业 | 页面语气 |
| --- | --- | --- | --- |
| 老虎型 | 进攻、机会、短线弹性 | 高收益、高波动、题材作业 | 直接、兴奋、强调机会 |
| 猫头鹰型 | 分析、逻辑、慢决策 | AI 解释充分、结构清晰的作业 | 理性、解释、强调逻辑 |
| 考拉型 | 稳健、防守、低波动 | 低回撤、高股息、分散组合 | 安心、克制、强调能抄得住 |
| 孔雀型 | 热门、社交、跟风灵感 | 明星作业、热门榜、讨论多的作业 | 轻松、热闹、强调正在发生 |

五道题只需要得到一个主人格，不需要复杂分数展示。结果页可以展示一个次级倾向，例如“考拉型，带一点猫头鹰”。

## 页面设计

### 1. `onboarding.html` 五道题测试

目的：让用户快速进入产品语境。

结构：

1. 顶部品牌：小金鲤 + “先测测你是哪种投资人格”。
2. 进度条：第 1/5 题。
3. 题目卡片：每题 3-4 个选项。
4. 底部按钮：下一题 / 查看结果。

题目方向：

1. 看到一只股票一天涨 8%，你第一反应是什么？
2. 账户回撤 10%，你通常会怎么做？
3. 你更愿意抄哪种作业？
4. 你看作业时最想先知道什么？
5. 你希望小金鲤怎么提醒你？

交互规则：

1. 每题选择后才能进入下一题。
2. 本静态版可用前端 JS 在本地计算人格。
3. 结果可写入 `localStorage`，用于首页展示默认人格。

### 2. `personality.html` 人格结果

目的：把测试结果转成推荐理由。

结构：

1. 大人格卡：动物名称、人格标题、三句解释。
2. 推荐作业类型：适合看什么、不适合看什么。
3. 风险提醒：一句话提醒，不做长篇教育。
4. 主按钮：去看适合我的作业。

示例：

```text
你是考拉型投资人
你更适合先看低回撤、分散配置、能长期拿得住的作业。
```

### 3. `discover.html` 新首页

目的：首页从“发现资讯”改成“看作业”。

首屏结构：

1. 顶部：小金鲤 + 当前人格标签。
2. 个性化提示：例如“你是考拉型，今天优先看稳健赚钱榜”。
3. 主模块：今天谁赚了大钱。
4. 三张作业卡：
   - 适合你的作业
   - 今日大赚作业
   - 低回撤赚钱作业

作业卡展示字段：

1. 作业名称 / 人物或机构。
2. 收益率。
3. 最大回撤。
4. 样本数或披露滞后。
5. 人格匹配理由。
6. AI 一句话解释。

弱化内容：

1. 交易热点不再作为首屏模块。
2. 热门主题可以变成“更多灵感”入口。
3. 投资运势/日签删除或降级为人格氛围小组件。
4. 工具箱不出现在首页主流程。

### 4. `copytrade.html` 作业池

目的：承接首页点击和完整浏览需求。

保留三类作业：

1. 大牛持仓。
2. 游资动向。
3. 私募持仓。

调整方向：

1. 顶部不再强调字段说明，而是强调筛选和排序。
2. 默认排序可以按当前人格变化：
   - 老虎型：收益率优先。
   - 考拉型：低回撤优先。
   - 猫头鹰型：AI 解释完整度优先。
   - 孔雀型：热度/讨论优先。
3. 卡片继续保留成功率、最大回撤、样本数、披露滞后。
4. 每张卡片增加“为什么推荐给你”的短句。

### 5. `portfolio.html` 作业详情 / AI 拆作业

目的：用户点开后直接理解这份作业。

首屏结构：

1. 作业名称。
2. AI 结论：一句话说明这份作业的核心逻辑。
3. 关键指标：收益率、最大回撤、持仓数量、主要风格。
4. 组合持仓：直接展示核心持仓和比例。
5. AI 拆解：为什么这样配。

要删除：

1. “抄作业前先问自己 3 个问题”不作为首屏。
2. 短期 / 中期 / 长期筛选不出现。

风险提示位置：

放在 AI 拆解后面，使用“这份作业不适合谁”的表达，而不是教育用户先反思。

### 6. `inspiration.html` 灵感页

目的：承接“找灵感/写灵感”。

结构：

1. 我的收藏作业。
2. 我的灵感笔记。
3. 小金鲤生成的观察问题。

静态版可先做轻量页面，不需要真实编辑器。支持点击“写下想法”弹出简单输入区域即可。

### 7. `ai_assistant.html` 小金鲤

目的：用户不确定时随时问。

入口：

1. 首页右下角浮球。
2. 作业详情里的“问小金鲤”按钮。
3. 灵感页里的“帮我整理这条想法”按钮。

问题建议：

1. 这份作业适合我吗？
2. 这个组合为什么这么配？
3. 它最大的风险是什么？
4. 如果我是考拉型，我该怎么看它？

### 8. `toolbox.html` 工具箱

处理方式：从主导航移除。

保留文件，但不在底部导航展示。后续如需运营，可以单独做成小程序、外部活动页或二级入口。

## 导航设计

当前：

```text
发现 / 抄作业 / 工具箱 / 我的
```

改为：

```text
首页 / 作业 / 灵感 / 我的
```

说明：

1. 首页：个性化作业流。
2. 作业：完整作业池。
3. 灵感：收藏、记录、写想法。
4. 我的：人格画像、关注、设置。

小金鲤 AI 不占底部导航，使用右下角浮球。

## 数据模型

静态版可用前端 JS 对象模拟数据。

作业对象建议字段：

```js
{
  id: "portfolio-koala-001",
  name: "睿远均衡",
  category: "bull",
  personalityFit: ["koala", "owl"],
  returnRate: "+18.6%",
  successRate: "63%",
  maxDrawdown: "-18.4%",
  riskLevel: "中低波动",
  sampleSize: "42次调仓",
  disclosureLag: "季报滞后45天",
  aiSummary: "用分散持仓降低波动，用消费和制造维持长期弹性。",
  fitReason: "适合想稳一点、但又不想完全放弃收益弹性的用户。",
  holdings: [
    { name: "宁德时代", weight: "9.5%", reason: "新能源核心仓位" }
  ]
}
```

人格对象建议字段：

```js
{
  id: "koala",
  name: "考拉型",
  headline: "先拿得住，再谈赚多少",
  preferredSort: "drawdown",
  recommendedCategories: ["bull", "private"],
  avoid: "高频短线和高回撤作业"
}
```

## 交互规则

1. 首次进入优先进入测试页。
2. 已有 `localStorage.personality` 时进入首页。
3. 首页内容根据人格展示不同推荐标题和排序。
4. 作业详情必须先给 AI 结论，再给持仓细节。
5. 点击“抄作业”在静态版中只模拟收藏/关注状态，不做真实交易。
6. 小金鲤 AI 使用模拟回复，围绕当前作业或人格生成解释。

## 测试与验证

新增或调整测试应覆盖：

1. 新页面存在：`onboarding.html`、`personality.html`、`inspiration.html`。
2. 底部导航不再包含工具箱。
3. 首页包含“今天谁赚了大钱”。
4. 首页弱化或移除“交易热点”首屏展示。
5. 作业详情包含 AI 结论。
6. 作业详情不包含“抄作业前先问自己 3 个问题”。
7. 作业详情不包含“短期 / 中期 / 长期”筛选。
8. 作业池仍保留三类作业切换。
9. 小金鲤入口仍可访问。

## 实施边界

本次改版建议分三步执行：

1. 信息架构改造：新增测试/画像/灵感页，调整底部导航。
2. 首页重做：从发现页改为个性化作业首页。
3. 作业详情重做：先结论、再持仓、再 AI 拆解。

每一步都应保持静态页面可打开、导航可跳转、测试通过。

## 待确认事项

1. 是否强制首次进入测试页，还是首页顶部提示用户去测试。
2. 四种动物人格是否固定为老虎、猫头鹰、考拉、孔雀。
3. 首页默认展示收益率榜，还是根据人格默认展示不同榜。
4. “灵感”页是偏笔记，还是偏收藏夹。

当前建议：

1. 首次进入强制测试，已有画像后直达首页。
2. 先固定四种动物人格。
3. 首页默认展示“适合你的人格榜”，同时提供“今日大赚榜”。
4. 灵感页第一版偏收藏夹 + 轻笔记。
