# 小金鲤静态复刻

这是从墨刀分享页本地化出来的静态 HTML 版本。

## 打开方式

直接打开 `index.html`，或用任意静态服务器指向本目录。

## 页面

- `index.html` / `discover.html`：发现
- `copytrade.html`：抄作业
- `toolbox.html`：工具箱
- `profile.html`：我的
- `ai_assistant.html`：AI 助手
- `portfolio.html`：组合清单
- `stock_detail.html`：股票详情

## 修改入口

每个页面都是独立 HTML，公共颜色在各页面头部的 Tailwind 配置里。底部导航和页面跳转已改成本地文件链接。
